SZÓKIRAKÓ JÁTÉK

Adott egy 4*9-es négyzetháló. A program megad egy - a négyzetháló feletti képhez
kapcsolódó - szót vagy mondatot, majd ennek betűit összekeverve elhelyezi ebben a
négyzethálóban.

   A játékos feladata az, hogy az összekevert betűkből kirajka a szót

      vagy mondatot a kijelölt helyen. A betűket vízszintes vagy függőleges irányban egyenként
lehet mozgatni

             

      Ha a betű újtát elállja egy másik betű, akkor az csak a másik betűig tolódik el.

Miután sikerült a betűket a megfelelő helyre mozgatni, a program ezt jelzi, ha viszont valamely
betűket felcseréltük akkor, akkor azokra színkiemeléssel hivja fel a
figyelmünket

a2 + b2 = c2

a + β + γ = 180°
